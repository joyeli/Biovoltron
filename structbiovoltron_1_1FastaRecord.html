Implicit conversion operator (converted to FastaRecord<!Encoded>)

  • If you have existing FastaRecord (or FastqRecord) object a = FastaRecord<false>, and you want to change a's encoding from std::string to biovoltron::istring, this operator allows you to implicitly convert you object to the !Encoded version. Example
    FastaRecord<false> a; // a is encoded by std::string
    FastaRecord<true> b = a; // b is encoded by biovoltron::istring (using
    // implicit conversion)
