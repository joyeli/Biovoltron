◆ tail_to_idx()

template<class Str >
int biovoltron::mirna::tail_to_idx ( const Str &  tail)

Classify a tail sequence into an index.

Template Parameters
StrString-like type supporting range-based for and substr (e.g. std::string).
Parameters
tailTail sequence string. Empty string denotes genome-matching.
Returns
category index:
  • 0: 'A' tail
  • 1: 'C' tail
  • 2: 'G' tail
  • 3: 'U' tail
  • 4: Other tail (mixed nucleotides)
  • 5: Genome-matching (empty tail)
  • -1: Invalid tail (contains non-ACGTU characters)
Note
if mutliple nucleotide types are present, returns 4 (Other).
