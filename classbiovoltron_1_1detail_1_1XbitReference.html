Detailed Description

template<std::size_t N, std::unsigned_integral Block>
class biovoltron::detail::XbitReference< N, Block >

A reference to an N-bit wide field within a block of memory.

Provides read/write access to N-bit values packed in a Block.

Template Parameters
NBit-width of each field.
BlockUnsigned integral type used as storage block.
